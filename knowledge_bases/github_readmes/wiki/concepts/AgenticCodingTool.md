## Summary
An agentic coding tool is a class of AI-powered developer assistance that autonomously executes tasks — such as writing code, running commands, and managing git workflows — rather than merely suggesting or explaining. Claude Code is a canonical example of this paradigm.

## Key Characteristics
- **Autonomous execution**: The tool takes actions (not just provides text output)
- **Codebase-aware**: Understands the full project context
- **Natural language interface**: Developers interact via plain English commands
- **Multi-step workflows**: Can chain operations like read → edit → test → commit
- **Tool use**: Integrates with terminal, filesystem, git, and external APIs
