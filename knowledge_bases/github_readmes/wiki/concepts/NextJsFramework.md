## Summary
Next.js is a React meta-framework that adds server-side rendering, static site generation, API routes, and full-stack capabilities to React's component model. It is the dominant framework in the React ecosystem, competing with Remix, Gatsby, and Astro. Its defining characteristic is tight integration of frontend and backend concerns within a single project structure.

## Key Characteristics
- **Full-stack in one project**: Combines frontend (React components), routing, and backend (API routes/serverless functions) in a single framework
- **Rust-based build tooling**: Integrates Rust-based JavaScript tooling (Turbopack) for fast incremental builds
- **Multiple rendering modes**: Supports SSR, SSG, ISR (incremental static regeneration), and client-side rendering per-route
- **Opinionated structure**: File-based routing, API routes, and automatic code splitting reduce configuration overhead
- **Community-driven**: Public roadmap, iteration plans, contribution guidelines, and "good first issues" for newcomers
