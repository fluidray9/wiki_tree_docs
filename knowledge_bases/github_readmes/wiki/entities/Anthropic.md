## Summary
Anthropic is an AI safety and research company that develops Claude Code, an agentic coding tool for developers. The company is known for its focus on building reliable, interpretable, and steerable AI systems.

## Key Facts
- Developer of Claude Code, an agentic terminal-based coding assistant
- Maintains official documentation at code.claude.com
- Hosts a Discord community for Claude Code developers
- Operates a public GitHub repository for Claude Code (github.com/anthropics/claude-code)
