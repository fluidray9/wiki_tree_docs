## Summary
Claude Code is an agentic coding tool by [[Anthropic]] that lives in the terminal, understands your codebase, and helps you code faster by executing routine tasks, explaining complex code, and handling git workflows through natural language.

## Key Facts
- Agentic coding tool: autonomously executes tasks rather than just suggesting
- Supports: MacOS/Linux (curl script, Homebrew), Windows (PowerShell, WinGet)
- npm installation is deprecated
- Has a plugin system for custom commands and agents
- Data collection: usage data, conversation data, bug feedback; NOT used for model training
- Community: Discord (anthropic.com/discord), GitHub issues
