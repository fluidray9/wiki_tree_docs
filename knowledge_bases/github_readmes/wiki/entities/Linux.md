# Linux

## Overview
Linux is the family of open-source Unix-like operating systems built around the Linux kernel. The kernel — the core system software managed by Linus Torvalds and a global community — handles hardware abstraction, memory management, process scheduling, and system calls. The operating system as a whole (Linux = kernel + GNU userland) powers everything from smartphones (Android) and servers to supercomputers and embedded devices.

## Key Facts
- **Kernel version**: Released incrementally; latest stable versions at kernel.org
- **License**: GPL v2 (kernel), mixed licenses for userland components
- **Kernel development**: Public mailing-list-based process at lore.kernel.org; patches submitted via email; DCO required
- **Distribution ecosystem**: Major distros include Ubuntu, Fedora, Debian, Arch, RHEL, openSUSE
- **Mobile**: Android uses the Linux kernel as its foundation
- **AI Coding Assistants**: The kernel has explicit documentation for LLMs (Documentation/process/coding-assistants.rst) requiring DCO compliance before any AI-assisted contributions
