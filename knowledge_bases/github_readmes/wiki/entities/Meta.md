## Summary
Meta (formerly Facebook) is the technology company that develops and maintains React, the popular JavaScript library for building user interfaces. Meta adopted a Code of Conduct for React contributors and oversees React's open-source development on GitHub.

## Key Facts
- Formerly known as Facebook; rebranded to Meta in 2021
- Develops and maintains React under open-source development (MIT license)
- Adopted a Code of Conduct for project participants (via code.fb.com)
- Security vulnerabilities in React must be reported privately via email to Meta's responsible disclosure program

## Connections
- [[React]] — the primary open-source project Meta maintains; the core library is the foundation of Meta's UI development ecosystem
- [[React-Native]] — Meta also develops React Native, extending the React programming model to mobile platforms
