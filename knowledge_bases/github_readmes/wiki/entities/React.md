## Summary
React is a JavaScript library for building user interfaces, maintained by Meta (formerly Facebook). Next.js is a framework that extends React with server-side rendering, routing, API routes, and full-stack capabilities. The relationship is one of foundation (React) and framework (Next.js).

## Key Facts
- Declarative JavaScript library for building component-based UIs
- Next.js extends React with server-side rendering (SSR), static site generation (SSG), and API routes
- Next.js claims to extend "the latest React features" as a core value proposition
- React's component model and hooks system are the foundation Next.js builds upon
