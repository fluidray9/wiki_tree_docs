## Summary
Vercel is a cloud platform for front-end frameworks and static sites, and the primary developer and maintainer of the Next.js open-source framework. Vercel sponsors Next.js development, hosts the official Next.js documentation and website, and runs the Next.js bug bounty program for responsible security disclosure.

## Key Facts
- Developer and maintainer of Next.js, the React framework for full-stack web applications
- Hosts the Next.js official documentation at nextjs.org and the Next.js Learn course
- Operates the responsible disclosure / bug bounty program for Next.js security vulnerabilities (responsible.disclosure@vercel.com)
- Maintains the next.js repository on GitHub, including iteration plans, contribution guidelines, and community governance
- Also known for its zero-config deployment platform used to host Next.js applications
