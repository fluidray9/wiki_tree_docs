## Summary
Next.js is a React framework for building full-stack web applications, developed and maintained by [[Vercel]]. It extends the latest React features and integrates powerful Rust-based JavaScript tooling to deliver fast builds. Used by some of the world's largest companies, Next.js is a foundational framework in the modern JavaScript/TypeScript ecosystem.

## Key Claims
- Next.js enables full-stack web application development by extending the latest React features
- It integrates powerful Rust-based JavaScript tooling (e.g., Turbopack's Rust compiler) for the fastest builds
- Used by some of the world's largest companies
- Community support is available via GitHub Discussions and a dedicated Discord server
- The project has a Code of Conduct that applies to all community channels
- Contributions are welcome; "good first issues" are labeled on GitHub for newcomers
- Security vulnerabilities must be reported privately via email to Vercel's responsible disclosure program, not as public issues

## Key Quotes
> "Next.js enables you to create full-stack web applications by extending the latest React features, and integrating powerful Rust-based JavaScript tooling for the fastest builds." — Next.js README
> "Do note that our Code of Conduct applies to all Next.js community channels. Users are highly encouraged to read and adhere to it to avoid repercussions." — Next.js README

## Connections
- [[Vercel]] — the company that develops and maintains Next.js, also the platform hosting the Next.js website and documentation
- [[React]] — Next.js is a framework that extends React, providing SSR, routing, and full-stack capabilities on top of React's component model
- [[Visual Studio Code]] — a complementary developer tool: VS Code is a cross-platform editor/IDE commonly used to write Next.js applications; Dev Containers and GitHub Codespaces are supported for Next.js development environments
- [[Rust]] — Next.js's build tooling is powered by Rust-based JavaScript tooling (Turbopack), contributing to its fast build performance
- [[Claude Code]] — Next.js applications can be developed with Claude Code's agentic terminal assistance; the React framework ecosystem is one area where AI coding tools can accelerate development
