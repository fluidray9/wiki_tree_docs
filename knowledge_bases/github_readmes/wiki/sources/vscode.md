## Summary
Visual Studio Code (VS Code) is a cross-platform code editor developed by [[Microsoft]], built on the open-source "Code - OSS" repository. It provides comprehensive code editing, navigation, debugging, and a rich extensibility model, updated monthly. The VS Code product combines Code - OSS with Microsoft-specific customizations under a traditional product license.

## Key Claims
- VS Code is a distribution of the open-source Code - OSS repository with Microsoft customizations
- It runs on Windows, macOS, and Linux, with monthly feature and bug-fix releases
- Insiders builds deliver the latest daily releases for early adopters
- The project publishes its roadmap, monthly iteration plans, and endgame plans publicly on GitHub
- Dev Containers and GitHub Codespaces are supported for containerized development environments
- Development from source requires at least 4 cores and 6 GB RAM (8 GB recommended)

## Key Quotes
> "Visual Studio Code combines the simplicity of a code editor with what developers need for their core edit-build-debug cycle." — VS Code README
> "This source code is available to everyone under the standard MIT license." — VS Code README

## Connections
- [[Microsoft]] — the developer and publisher of VS Code and the Code - OSS open-source project
- [[ExtensibleIDE]] — VS Code's extensibility model is a defining characteristic, enabling rich language support and custom integrations via the extension API
- [[DevContainers]] — VS Code supports Dev Containers and GitHub Codespaces for cloud-based, reproducible development environments
- [[Claude Code]] — complementary developer tools: VS Code provides the GUI-based editor and debugging environment, while Claude Code provides agentic terminal-based coding assistance; both support extensibility and cross-platform use
